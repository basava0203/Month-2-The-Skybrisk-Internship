# Month-2-The-Skybrisk-Internship
# 🚢 Titanic Survival Prediction - Machine Learning Project

## 📌 Project Overview

This project implements a comprehensive machine learning solution to predict passenger survival on the Titanic. Using the classic Titanic dataset, we explore various ML algorithms, perform extensive feature engineering, and optimize model performance to achieve reliable predictions.

### 🎯 Key Objectives
- Build predictive models to classify passenger survival
- Identify the most influential factors in survival outcomes
- Compare multiple ML algorithms with hyperparameter tuning
- Optimize model training time while maintaining accuracy

---

## 📊 Dataset Information

The dataset contains **891 passenger records** with 12 features:

| Feature | Description |
|---------|-------------|
| PassengerId | Unique passenger identifier |
| Survived | Target variable (0 = Died, 1 = Survived) |
| Pclass | Passenger class (1st, 2nd, 3rd) |
| Name | Passenger name |
| Sex | Gender |
| Age | Age in years |
| SibSp | Number of siblings/spouses aboard |
| Parch | Number of parents/children aboard |
| Ticket | Ticket number |
| Fare | Passenger fare |
| Cabin | Cabin number |
| Embarked | Port of embarkation (C = Cherbourg, Q = Queenstown, S = Southampton) |

**Survival Rate:** 38.4% (342 survivors out of 891 passengers)
